# ARM_Assembly_Projects
ARM assembly language projects using the ARMLite simulator at: https://peterhigginson.co.uk/ARMlite/
- Stick Game: player goes against PC, the loser is the last player to take sticks out of the pile. Each player has a turn and can take out 1-3 sticks from pile. After taking out the chosen amount of sticks the turn goes to the other player.
- Snake Game V1: classic snake game.
- Snake Game V2: classic snake game with some add-ons: 1- player can make 2 mistakes before losing, instead of 1; 2- there are 4 apples at the same time on the set; 3- black points are bombs, if snake eats the bomb the player loses 1 health point.
- Games are all in portuguese, but can easily be played by anyone.
- All codes can be run at: https://peterhigginson.co.uk/ARMlite/
